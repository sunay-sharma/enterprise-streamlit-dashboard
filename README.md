# Enterprise Data Analytics Dashboard

This repository contains a professional Streamlit dashboard built for a final data analysis project.

## Features
- Interactive visualizations
- Correlation & trend analysis
- Category comparisons
- Executive KPI dashboard
- Light & Dark mode

## Files
- app.py – Streamlit application
- dataset_cleaned_full.csv – Cleaned dataset
- requirements.txt – Python dependencies

## Deployment
The dashboard is deployed using Streamlit Community Cloud.
